Running

You'll need jupyter_console (soon to be released) in order to launch consoles. Sidecar also attaches to notebook kernels.

Start up a console:

$ jupyter console

Fire up sidecar (from within the cloned and built directory):

$ npm run start

Sidecar will open as many display areas as there are kernels running, which means if you run

jupyter console

in separate terminals, sidecar views will pop up automagically.

Building

This package requires node or iojs, zmq, and jupyter_console (soon to be released).

Additionally, you'll need node-gyp:

npm install -g node-gyp

After cloning this repository and cding into the directory, run this:

$ npm run build
